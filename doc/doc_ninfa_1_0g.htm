2. FREEBASIC INSTALLATION

The installation of the Ninfa3D Lite is very simple, but you must follow certain steps:
1. Copy the file that you will find in the folder "inc", in another folder with the same name that exists in FreeBASIC's directory.
2. Copy all the libraries there are, in the folder "dll" , in the folder where you have your own project.
3. Inserts in the first line of the code of your project the line #include "ninfa3d.bi".
If any mistake goes out for you on having initiated your project or the examples, it is very possible that not this one installed in your system the library OpenAL v1.1.
If that is the case, run the file "OpenAL-Install-.exe" links at the outside of this folder.

COMMAND
  CreateCamera

SYNTAX
  Freebasic:
    CreateCamera( ) As nCAMERA
  

PARAMETERS
  • None.

DESCRIPTION
  It creates a simple camera.
  To move this camera, you have to use the commands of movement.

EXAMPLE
  Freebasic:

 
     Dim As nCAMERA camera = CreateCamera( )




COMMAND
  CreateCameraFPS

SYNTAX
  Freebasic:
    CreateCameraFPS( srot As single, smov As single ) As nCAMERA

PARAMETERS
  • srot - Velocity of rotation.
  • smov - Velocity of movement.

DESCRIPTION
  It creates a camera very similar to the games FPS.
  This camera only can be rotated by the mouse and move with the arrows of the keyboard. To define the keys of control of the keyboard, use the command.

EXAMPLE
  Freebasic:

 
     Dim As nCAMERA camera = CreateCameraFPS( 100.0, 0.5 )




COMMAND
  CreateCameraMAYA

SYNTAX
  Freebasic:
    CreateCameraMAYA( srot As single, szom As single, smov As single ) As nCAMERA

PARAMETERS
  • srot - Velocity of rotation.
  • szom - Velocity of zoom.
  • smov - Velocity of movement.

DESCRIPTION
  It creates a camera very similar to that of program of modeling Maya.
  The camera is controlled totally by the mouse:
  • Left Button - Rotated the camera.
  • Centre Button - Zoom.
  • Right Button - Moves the camera.

EXAMPLE
  Freebasic:

 
     Dim As nCAMERA camera = CreateCameraMAYA( 100.0, 100.0, 100.0 )
